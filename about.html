We use a simple flashcard based approach to learning to read, write and understand the given writing system. There are 2 components of the app:

  1. Introduction to the writing system to help beginners with learning the fundamentals
  2. Practice - Helps you learn to read and write the given character

The app is not a dictionary, the user is supposed to use a separate dictionary(like pleco for Chinese)

The app also supports limited features for learning to write using other writing systems like Latin, Hiragana, Katakana, Cyrillic, Georgian, Armenian, Arabic and other writing systems

Practice, the main feature

Practice mode is the main feature of the app, it's a flashcard based system for learning characters. When you add a character to your dictionary it will be in the learning list, then when you start practicing it will first be introduced fully a couple of times, then as you get better at drawing it, it components will be removed until you can recall how to write the character in its entirety

This is also useful for people new to languages with a script that's unknown to the learner

Introduction to the writing system

The introduction feature starts on first login in the app and is designed to primarily help beginners with learning the writing techniques in their specific script as well as how to read and tips for remembering characters

For CJK languages that would be learning how to write dynamically, learning about radicals and tips for remembering the characters.

All languages provided also get a brief phonetics guide to help with pronunciation of letters and words

Non-CJK languages also have guides for cursive

The story of the app's development

The founder of the project is Stanislav Vasilev(Madman10K). At the time in 2023 he was learning Chinese and German, his mother tongue is Bulgarian, which is written in Cyrillic, while his second language is English, which is written in a modified version of the latin alphabet.

While trying to learn Chinese, a problem kept occurring: because he didn't write using Cangjie or Wubi and instead used a Pinyin based input method, he found it easy to learn how to read characters but not how to write them, or at least that was in the beginning of his Chinese learning journey. When a learner initially starts learning Chinese characters, it may be easy to remember to read them at first, but at some point many similar characters enter the person's memory and without knowing how to write them, the learner has almost no ability to tell components apart and therefore differentiate characters, not to mention reading handwritten characters.

For this very reason, Youyin was created. Initially as a platform for learning to write Chinese characters, Youyin now supports many more scripts including Hiragana, Katakana, Hangul, Cyrillic, Latin, Greek, Georgian, Armenian, Arabic and more.
